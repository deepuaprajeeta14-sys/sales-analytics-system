
# Sales Analytics System

This project processes messy sales transaction data, cleans invalid records, and generates basic sales analytics using Python.

## Project Structure

- main.py – Entry point of the application

- utils/file\_handler.py – Reads sales data with encoding handling

- utils/data\_processor.py – Cleans, validates, and analyzes data

- data/sales\_data.txt – Input sales data file
